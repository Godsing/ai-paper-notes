# Scoring Rubric

## Industrial Value (1-5)

- **5**: 对产品、Agent、部署、推理、评测或训练流程有明显短期价值，值得团队尽快试验。
- **4**: 有较强应用潜力，值得工程或研究团队关注。
- **3**: 有参考价值，但更偏中期或局部启发。
- **2**: 研究意义大于落地意义。
- **1**: 当前几乎没有明显工业价值。

优先加分项：agent workflow、tool use、inference efficiency、deployment、evaluation、data efficiency、robustness。

## Readability (1-5)

- **5**: 问题直观、方法清楚、实验容易理解。
- **4**: 大体清晰，少量技术门槛。
- **3**: 可以读懂，但需要一定背景。
- **2**: 结构复杂或实验解释成本高。
- **1**: 很难快速读懂。
